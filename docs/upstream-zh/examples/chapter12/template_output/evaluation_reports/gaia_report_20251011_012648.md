# GAIA Evaluation Report

## Summary

This is a sample GAIA evaluation report generated by the Chapter 12 examples.

## What the Report Shows

- Dataset level and sample count.
- Exact-match and partial-match results.
- Agent answers and reference answers.
- Output files from the evaluation run.

## Notes

GAIA is a gated benchmark. Request access on Hugging Face and set `HF_TOKEN` before running real evaluations.
